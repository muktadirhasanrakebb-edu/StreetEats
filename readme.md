# 🍔 StreetEats

**StreetEats** is a location-based web application designed to connect food lovers with local street food vendors. It helps users discover nearby street food, explore vendors, and enjoy real-time food experiences.

---

## 🚀 Features

* 📍 **Location-Based Discovery**
  Find nearby street food vendors using geolocation.

* 🍜 **Vendor Listings**
  Browse different street food vendors and their offerings.

* 🔎 **Search & Filter**
  Easily search food items or vendors.

* 🧑‍💻 **User-Friendly Interface**
  Clean and responsive frontend for smooth user experience.

* 🗂️ **Backend Integration**
  Handles data, APIs, and business logic.

## 🛠️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/muktadirhasanrakebb-edu/StreetEats.git
cd StreetEats
```

### 2️⃣ Setup Backend

```bash
cd backend
npm install
npm start
```

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm start
```
